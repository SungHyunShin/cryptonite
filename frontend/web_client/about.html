About us

We are computer science students from the University of Notre Dame with a server based project for the class Programming Paradigms. This project is a web and smart-watch client that helps users track their crypto-currency wallets and make smarter investments in the cryptocurrency field. It can display the hottest and coldest currencies, simulate a user's specific crypto-currency wallet, and recommend a crypto-currency based on user preference for volatile or stable currencies.

Andy Shin, Nicholas Marcopoli, Luke Song, Austin Sura.
